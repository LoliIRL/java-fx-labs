# JavaFX Лабораторная работа №10

## Описание проекта
Этот проект содержит два JavaFX приложения:
1. **Калькулятор** - полнофункциональный калькулятор с поддержкой клавиатуры
2. **Geometry Drawing App** - приложение для рисования и управления геометрическими фигурами

## Структура проекта
src/
├── main/
│ ├── java/
│ │ ├── calculator/ # Приложение калькулятора
│ │ ├── geometry/ # Приложение для рисования фигур
│ │ └── geometry/geometry2d/ # Пакет геометрических фигур
│ └── resources/
│ ├── calculator/ # FXML файлы калькулятора
│ └── geometry/ # FXML файлы geometry приложения
